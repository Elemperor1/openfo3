How to use this checklist:

- This file is the exhaustive Fallout 3 parity inventory for `openfo3`.
- It is intentionally flatter and more exhaustive than the working roadmap.
- Use `docs/openfo3-gameplay-roadmap.md` for execution order and near-term build phases.
- Use `docs/openfo3-plan.md` for the original bootstrap/history context around the `openfo3` slice.
- Not every item here belongs in the next milestone; many are later-phase parity or compatibility work.
- Treat this file as the completeness ledger, not as the sprint board.
- If the roadmap and checklist seem to disagree on timing, prefer the roadmap for sequencing and this file for coverage.

You’re not going to get a literally perfect “every hidden edge case Bethesda ever shipped” list from anybody without months of reverse-engineering, but this is the closest thing to a full Fallout 3 engine-parity checklist I’d hand to a team starting an OpenMW-style fork.
